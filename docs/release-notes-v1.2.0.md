## DownloadMaster v1.2.0

### What's new

#### Convert tab
- **Video Convert** - convert, optimize, GIF, and frame extraction (FFmpeg) with drag-and-drop and results panel
- **Image Convert** - single file or folder batch; PNG, JPEG, WebP, BMP, TIFF, GIF, TGA, DDS
- **PNG palette optimization** - reduce colors (16-256), optional dither, transparency preserved, strong file-size savings
- Downloads queue hidden on Convert tab (conversion uses its own results list)

#### Files tab
- Direct HTTP downloads with multi-connection support
- Pause, resume, and `.part` state for interrupted downloads
- Public Google Drive sharing links supported

#### Instagram & app (from 1.1.x)
- Profile browse (@username): stories, highlights, posts, carousels with thumbnails
- Cookies: sync from Chrome/Edge, paste from DevTools, or clear
- Copy failure details from the download queue
- Dark/light themes, English and Arabic (RTL)

### Video downloads
- 1000+ sites via yt-dlp (YouTube, X/Twitter, playlists, fetch preview, queue progress)
- Open folder / play when complete, per-download save path

### Install
Windows 10+ x64. Extract the zip and run `DownloadMaster.exe`. No .NET runtime required.

Bundled **yt-dlp** and **FFmpeg** are included under `tools\`.
