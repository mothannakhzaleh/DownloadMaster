## DownloadMaster {{VERSION}}

### What's new

#### Convert tab — Audio & Speech
- **Audio Convert** — convert or optimize audio; extract audio track from video files
- Output formats: MP3, WAV, M4A, FLAC, OGG, MP2, AMR, iPhone ringtone (M4R)
- Quality presets (Economy through Best); same-folder or custom output folder
- **Speech** — text-to-speech using Windows built-in voices
- Preview with speed and volume controls; export WAV, MP3, or M4A
- Drag-and-drop, probe panel, and per-job results list on both sub-tabs

#### UI & localization
- Dark theme polish for Convert sub-tabs and format chips
- Compact nested tab bar for Video / Image / Audio / Speech
- English and Arabic strings for all Convert views and result columns
- Updated About dialog with full feature list

#### Convert tab (from 1.2.x)
- **Video Convert** — convert, optimize, GIF, and frame extraction (FFmpeg)
- **Image Convert** — single file or folder batch; PNG palette optimization (16-256 colors)
- Downloads queue hidden on Convert tab

#### Files tab
- Direct HTTP downloads with multi-connection support
- Pause, resume, and `.part` state for interrupted downloads
- Public Google Drive sharing links supported

#### Instagram & app
- Profile browse (@username): stories, highlights, posts, carousels with thumbnails
- Cookies: sync from Chrome/Edge, paste from DevTools, or clear
- Copy failure details from the download queue
- Dark/light themes, English and Arabic (RTL)

### Video downloads
- 1000+ sites via yt-dlp (YouTube, X/Twitter, playlists, fetch preview, queue progress)
- Open folder / play when complete, per-download save path

### Install
Windows 10+ x64. Extract the zip and run `DownloadMaster.exe`. No .NET runtime required.

Bundled **yt-dlp** and **FFmpeg** are included under `tools\`.
