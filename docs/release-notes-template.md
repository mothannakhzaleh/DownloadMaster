## DownloadMaster {{VERSION}}

### What's new
- **Instagram tab** - browse profiles (@username): stories, highlights, posts, and carousels with thumbnails
- **Smart selection** - select all, expand/collapse sections, linked carousel/highlight checkboxes
- **Direct downloads** - photos, reels, and carousel slides via Instagram API (not only yt-dlp)
- **Cookies** - sync from Chrome/Edge, paste from DevTools, or clear saved session
- **Diagnostics** - copy failure details from the download queue for troubleshooting
- **UI** - dark/light themes, English and Arabic (RTL), collapsible download panel

### Video downloads (unchanged core)
- 1000+ sites via yt-dlp (YouTube, X/Twitter, playlists, fetch preview, queue progress)
- Open folder / play when complete - per-download save path

### Install
Windows 10+ x64. Extract the zip and run `DownloadMaster.exe`. No .NET runtime required.

Bundled **yt-dlp** and **FFmpeg** are included under `tools\`.
